on using bsim.py:
  1. creating input data.
  bsim.py needs 'all...' files, hl files, and a locates
  file as input data. To access the input data, users also need to save them
  in correctly named folders. That is, 'all...' files
  need to be in a folder named 'all', hl files in a folder named 'hl',
  and locates files in a folder named 'locates'.

    a) creating the necessary directory structure
    Users should first run gen_dir.py, which creates a directory named 'data'
    in a root directory specified by the parameter '--dir'. This script will
    also create directories in the 'data' directory titled 'all', 'all_graphs',
    'hl', 'locates', 'raw', 'blotter', and 'opt'.

    For example, the following command creates the aforementioned directories:
    python3 gen_dir.py --dir=/Users/aaronhersch/Desktop/salamander

    b) creating 'all...' files.
    Users can generate 'all...' files by running gen_hl.py on raw data. To get
    raw data, users go to the raw_data folder in the drive and download
    raw data from the period of interest. Make sure to download only the
    spearmint colored raw data folders since only those folders
    are fully updated. Users also need to download raw data a year
    before the period of interest since gen_hl.py uses prior data. Once
    downloaded, raw data need to be put under the folder named 'raw' previously
    generated using gen_dir.py. To run gen_hl.py on raw data, users need to
    specify certain commands, the instructions to which can be accessed
    by putting '-h' or '--help' in the command line.

    For example, the following command creates 'all...' files from 20100630
    to 20130630: python3 gen_hl.py --start=20100630
    --end=20130630 --dir=C:/Users/aaronhersch/Desktop/salamander

    c) getting locates data
    Go to borrow_data folder in the drive, download borrow.csv, and put it into
    the locates folder generated by gen_dir.py.

    d) creating hl alpha signal
    Users need to run gen_alpha.py on 'all...'files. In the command
    line, users specify '--dir', the directory to the *folder* of
    'all...' files, '--start', the starting date of alpha signals, and
    '--end', the end date of alpha signals. The path passed as '--dir' should
    be the same as the path passed as '--dir' in gen_dir.py.

    For example, to generate hl alpha signal from 20100630 to 20130630,
    users could put: python3 gen_alpha.py --start=20100630
    --end=20130630 --dir=/Users/aaronhersch/Desktop/salamander

  2. running bsim.py
  To run bsim.py, users need to specify the following commands:
  '--start', the starting date of the simulation period; '--end', the end date
  of the simulation period; '--fcast', alpha signals; '--dir', the root directory.
  Each alpha signal needs to be separated by ',', and within each alpha signal,
  users need to specify the name, the multiplier, and the weight of the alpha signal,
  separating them by ':'. Additionally, the root directory should be the path to
  the 'data' folder generated by gen_dir.py, and should be the same as the '--dir'
  argument given in gen_dir.py.

  For example, to run bsim.py on the period between 20130101 and 201300630,
  users could put: python3 bsim.py --start=20130101
  --end=20130630 --dir=/Users/aaronhersch/Desktop/salamander --fcast=hl:1:1
